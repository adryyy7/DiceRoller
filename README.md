# DiceRoller

App Android che simula il lancio di un dado a 20 facce.

## Modifica apportata
Il progetto originale simulava un dado a 6 facce con immagini.
Ho modificato l'app per usare un dado a 20 facce, mostrando
il risultato come numero in un riquadro colorato.
